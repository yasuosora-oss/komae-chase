# 狛江チェイス（KOMAE CHASE）

東京都・狛江市を舞台にした、ボードゲーム「シティチェイス」アレンジ版のブラウザゲームです。
8ビット風のボクセル（積み木）グラフィックで、立体視点（3D）と真俯瞰（真上）を切り替えながら、
**犯人1人 vs 警察3人**の追いかけっこを楽しめます。

> これは「動く試作（最小版）」です。まずは遊べることを優先しています。ここから少しずつ良くしていきます。

---

## 遊び方

**2〜4人**で、1台の端末を回しながら遊びます。1人が **犯人**、残りが **警察**。
スタート画面で **参加人数（2〜4人）** を選びます。1人が犯人、残りが警察なので、**警察コマ＝参加人数−1**（2人→警察1／3人→警察2／4人→警察3）。
（「ひとりで遊ぶ（犯人＝AI・警察3人）」も選べます）

- 街は **25か所のビル**（それぞれ **看板** で名前が出ます）と、その間の **道路の交差点** でできています。

### 準備（シティチェイス通り）
1. **犯人**が隠れるビルを1つ選ぶ（他の人は画面を見ない）
2. 画面を渡し、**警察**が3機の置き場所（道路の交差点）を決める

### 各ターン
- **警察の番**：警察を1人選び、行動を1回。
  - 緑のマス（交差点）をタップ＝**移動**（タテヨコ1マス、斜め不可）
  - 黄色のビルをタップ＝**確認（さがす）**。今いる交差点の **まわり4つのビル** を調べられる
  - 確認したビルに犯人がいれば **発見＝警察の勝ち！**
- 3人すべて行動すると **犯人の番**：犯人は隣のビルへ1マス逃げ、通った所に痕跡を残す（同じビルへは戻れない）。

### 痕跡（手がかり）
- **痕跡は確認するまで見えません**。警察が確認したビルに痕跡があれば、そこで初めて **★マーク** で表示されます。
- ★の色：**1ターン目＝黄／6ターン目＝赤／その他＝グレー**、ターン番号付き（本家のコマと同じ）。

### 勝敗・答え合わせ
- **警察の勝ち**：確認で犯人を発見／犯人が動けなくなる。 **犯人の勝ち**：**11ターン**逃げ切る。
- ゲーム終了後、「**盤面で答え合わせ**」を押すと、犯人が通った全ルート（番号付きの★＋黄色いライン）を確認できます。
- 各ビルには **看板** で名前が出ます。**俯瞰視点**にすると盤面全体が見やすく、タップもしやすいです。
- 画面上のボタン：`3D視点`／`俯瞰視点`の切り替え、`？`（あそびかた）、`最初から`。

## あそぶ方法

### 方法1：パソコンでそのまま開く（いちばん簡単）
`index.html` を **ダブルクリック** すると、ブラウザ（Chrome/Edge/Safari）で開いて遊べます。
インターネット接続が必要です（描画ライブラリをネット経由で読み込むため）。

### 方法2：iPad など、みんなで遊ぶ（GitHubで公開する）
GitHub Pages を使うと、URLを開くだけで誰でも・iPadでも遊べるようになります。手順は下記。

---

## iPadで遊べるようにする（GitHub Pages 公開手順）

専門知識は不要です。順番にやればOK。

1. **GitHubのアカウント**を作る（無料）: https://github.com
2. 画面右上の「＋」→ **New repository**（新しい置き場所）を作る。
   - 名前：例 `komae-chase`
   - 公開範囲：**Public**（公開）
   - 「Create repository」を押す
3. できたページの「**uploading an existing file**」リンクを押し、
   この `index.html` と `README.md` を **ドラッグ＆ドロップ** でアップロード →「Commit changes」。
4. リポジトリ上部の **Settings** → 左メニューの **Pages** を開く。
   - 「Build and deployment」の Source を **Deploy from a branch** に。
   - Branch を **main** ・フォルダ **/(root)** にして **Save**。
5. 1〜2分待つと、`https://（あなたのID）.github.io/komae-chase/` のURLが表示されます。
6. そのURLを **iPadのSafariで開く** と遊べます。
   - Safariの「共有」→「**ホーム画面に追加**」で、アプリのように起動できます。

## 中身のこと（さわってみたい人向け）

- ファイルは `index.html` の **1つだけ**。HTMLとJavaScriptがまとまっています。
- 3D描画には [Three.js](https://threejs.org/)（無料・世界中で利用）を使用。
- 25か所の場所は `index.html` 内の `PLACES` という一覧で定義しています。
  名前や並び順を書き換えると、盤面が変わります。

### いまの試作の「割り切り」ポイント（今後の改善候補）
- 盤面の配置は一覧の順番どおり（本家の正確な配置は今後あわせ込み）。
- 犯人AIは「警察から遠ざかる」シンプルな思考（今後かしこく）。
- 痕跡コマは常に盤面に見える形（本家のターン番号表示などは今後）。
- 警察の行動は「移動」か「確認」の1回ずつ（本家のニュアンスに合わせ微調整可）。
- キャラ・建物の見た目はさらに作り込み可能。
